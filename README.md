# Personal Dashboard

A modern, interactive **Personal Dashboard** that allows users to:

- View a dynamic greeting based on the time of day.
- Check the current date and time.
- Get a daily quote and fetch new quotes on demand.
- Retrieve real-time weather information for any city.
- Manage a **Todo List** with persistent storage using `localStorage`.

## Features

- **Dynamic Greeting**: Updates based on time of day.
- **Time & Date Display**: Always shows current local time.
- **Quote of the Day**: Fetches and updates quotes with a button.
- **Weather Updates**: Fetches real-time weather info for any city.
- **Todo List**:
  - Add, complete, and delete tasks.
  - Tasks persist in `localStorage`.
  - Notifications for actions (add/delete).
  - Empty state handling.

## Tech Stack

- **HTML5 & CSS3** for structure and styling.
- **Vanilla JavaScript** for interactivity and localStorage management.
